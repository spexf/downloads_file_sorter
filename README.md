# File Sorter
Simple tool to help you manage your downloads directory automatically. This tool will sort your file into a directory by its extension.
### How to run
1. If this is your first time using it, run the create_folder.py first. This py is used to create necessary directory to store the file.
2. Modify main.py file to point your downloads directory.
3. If step 1 completed without error, you can run main.py to sort the file in download folder.
4. If you want to add another format file, you can modify the json file and then do the first step again.
